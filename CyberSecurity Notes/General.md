# Security Controls

- All controls fit in these types categories:
    - Preventive
    - Deterrent 
    - Detective
    - Corrective
    - Compensative 
    - Directive

## Technical Controls
- Use some type of technical system. E.G. Firewalls, Antivirus, etc.

## Managerial Controls
- Security Policies, SoP

## Operational Controls
- Security Guards

## Physical Controls
- Gates, fences, Keycards
